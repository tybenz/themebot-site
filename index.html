Autonomous theming made simple

Build a theming application for any CSS framework with a simple, but comprehensive javascript API

Get it!

Why?

When using a CSS framework, it's nice to have a fast way to customize the look and feel of their new app. ThemeBot provides the developers of those frameworks what they need to implement an application that allows for quick customization of the framework's theme

How?

ThemeBot provides some basic UI controls common among theming application so users can preview what their theme looks like as they go. It also provides the necessary CSS parsing to provide valid, production-ready CSS.

Possibilities

ThemeBot was designed for use with large-scale CSS frameworks. But who says it has to be built for multiple users? ThemeBot works great for individual client work too. A couple extra minutes configuring the app, and your clients can skin their own site to their hearts content

Demo

See an example of ThemeBot at work. I'm using the CSS framework Skeleton applied to a simple blog site. Quickly theme our demo blog site to get an idea of what ThemeBot can do!
